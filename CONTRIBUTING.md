# Contributing

Obrigado por considerar contribuir com este projeto! Antes de abrir PRs, por favor:

- Forque o repositório e crie um branch com um nome descritivo.
- Faça alterações pequenas e focadas (um propósito por PR).
- Rode os checks locais:

```bash
npm install
npm run lint
npm test
npm run build
```

- Escreva um comentário claro no PR descrevendo a mudança.
- Mantenha o padrão de código e formatação (ESLint + Prettier).
